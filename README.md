  <h2 align="center">Baralho Aleatório</h2>

  <p align="center">
Este projeto envolve a realização de uma requisição GET a uma API utilizando a função fetch, que é uma forma de fazer solicitações de recursos em uma rede. O método GET é usado para obter informações de um servidor web. No contexto deste projeto, foi desenvolvido um sistema que cria um baralho e retira uma carta aleatória desse baralho para exibi-la na tela. Além disso, há um botão que permite ao usuário obter uma carta aleatória com um simples clique.<br /><br />
